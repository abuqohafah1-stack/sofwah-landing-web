## Running locally
PHP 8.3 + Node are present. To run the actual Laravel app locally you also need **Composer**
+ the standard Laravel 11 skeleton (`bootstrap/`, `artisan`, `config/`, `.env`) generated by
`composer create-project`; these app files overlay it. Frontend build works standalone:
`npm install && npm run build`.
